# Dex 和 Clay 深度分析

> 调研日期：2026-04-16
> 产品类别：第一梯队个人 CRM 代表

---

## 一、Dex（getdex.com）

### 1.1 基本信息

| 项目 | 详情 |
|------|------|
| 定位 | 个人人脉维护工具，LinkedIn 重度用户首选 |
| 创始时间 | 约 2019 年（创始人自称运营 7 年） |
| 平台 | iOS、Android、macOS、Windows、Web、Chrome 浏览器插件 |
| 评分 | JustUseApp 4.4/5，G2 5.0/5（仅 1 条） |

### 1.2 功能清单

| 功能 | 详情 |
|------|------|
| LinkedIn 同步 | Premium 2,500 联系人，Professional 9,000 联系人 |
| Gmail/Outlook 集成 | 自动导入邮件往来联系人 |
| 日历集成 | Google Calendar 同步，会前 AI 简报 |
| 浏览器插件 | 在 LinkedIn 页面一键添加联系人 |
| 名片扫描 | 移动端支持 |
| 联系人分组 | 标签 + 自定义字段 |
| 保持联系提醒 | 可设置联系频率，推送提醒 |
| AI 会前简报 | Premium 及以上包含，自动生成对方背景摘要 |
| Dex Copilot | 自然语言查询联系人信息 |
| 邮件群发（Mail Merge） | 仅 Professional 计划 |
| 双向联系人同步 | Google/Apple 通讯录同步 |
| 地图视图 | 按地理位置查看联系人 |
| 笔记 | 记录交互详情 |
| Zapier + API 接入 | 仅 Professional 计划 |

### 1.3 定价

| 计划 | 年付月价 | 月付 | 核心差异 |
|------|----------|------|----------|
| Free | $0 | $0 | 基础体验，功能有限 |
| Premium | $12/月 | $20/月 | LinkedIn 2,500 联系人，AI 会前简报，全平台 App |
| Professional | $20/月 | 约 $30/月 | LinkedIn 9,000 联系人，Mail Merge，Outlook，Zapier，API |

学生优惠：本科生 50% 折扣，MBA/法学院有团体折扣。

### 1.4 用户评价

**好评（来自 Capterra、Software Advice）：**
- "最好的个人 CRM"
- "自动化同步 LinkedIn、Google 等工具，省时"
- "提醒功能确实改变了行为，不只是噪音"

**差评（来自 JustUseApp，33.2% 负面体验）：**
- **Bug 多**：按钮失效、页面空白、保存慢
- **同步慢**：跨设备同步要几分钟，Mac 端加载卡顿
- **导入问题**：通讯录导入后字段不匹配，2000 个联系人只导入 2 个电话和邮箱
- **客服差**：两次求助无回应
- **线下社交不友好**：活动现场快速捕获联系人不够快
- **年付压力**：优惠价需年付，月付价格翻倍

### 1.5 优缺点总结

| 优点 | 缺点 |
|------|------|
| 全平台覆盖（含移动端） | Bug 多，同步不稳定 |
| LinkedIn 集成深度行业第一 | 导入功能问题多 |
| AI 会前简报实用 | 年付制，月付贵 |
| 定价相对合理（$12/月） | 客服响应慢 |
| 学生优惠好 | 非团队工具，无协作 |
| 7 天免费试用，无需信用卡 | 线下场景不友好 |

## 二、Clay / Mesh（clay.earth → me.sh）

### 2.1 基本信息

| 项目 | 详情 |
|------|------|
| 定位 | 投资人/VC/创始人的人脉管理，被动数据丰富 |
| 创始时间 | 2019 年（创始人 Matthew Achariam、Zachary Hamed） |
| 融资 | 种子轮 $800 万（Forerunner Ventures、General Catalyst） |
| 收购 | 2024 年 6 月被 Automattic（WordPress 母公司）收购 |
| 品牌变更 | 2024 年重命名为 Mesh（me.sh），clay.earth 仍可用 |
| 平台 | iOS、Mac、Web（**无 Android**） |
| 注意 | Clay.com 是完全不同的 B2B 数据丰富平台（$149/月起），与 clay.earth 无关 |

### 2.2 功能清单

| 功能 | 详情 |
|------|------|
| 自动联系人丰富 | 从 Gmail、Outlook、LinkedIn、Twitter、iCloud 自动拉取数据 |
| 关系强度评分 | AI 评估关系亲密度 |
| 日历上下文 | 自动关联会议与联系人 |
| iMessage 集成 | macOS 端（需全磁盘访问，可选） |
| WhatsApp 集成 | 收购后新增 |
| 自然语言搜索 | "去年在 Austin 见过谁？" |
| 保持联系提醒 | 自定义联系频率 |
| 自定义 URL | t.me/username 式联系人链接 |
| Zapier 集成 | 自定义工作流 |
| 团队共享网络 | Team 计划支持最多 4 人 |
| 高级数据丰富 | Team 计划特有 |
| Chrome 插件 | 从任何网页一键添加联系人 |

### 2.3 定价

| 计划 | 价格 | 联系人限制 | 说明 |
|------|------|-----------|------|
| Free | $0 | 最多 1,000 人 | 基础功能，适合试水 |
| PRO | $10/月（年付） | 无限 | 全功能解锁 |
| Team | 定制报价 | 最多 4 用户 | 含高级数据丰富、共享网络 |

### 2.4 用户评价

**好评：**
- "根本性地改变了我对联系人管理的思考方式"
- "被动数据丰富是杀手级功能，不用手动输入"
- "免费 1,000 联系人是最好的入门方案"
- "iOS App 评价很好"

**差评/隐忧：**
- **收购后沉寂**：Automattic 收购后社交媒体活动大幅下降，博客停更
- **无 Android App**：Android 用户完全无法使用
- **无移动 App（folk 对比中提到的）**：部分用户指出 Clay 没有独立移动 App
- **客服无回应**：有用户联系团队未获回复
- **功能限制**：自定义 URL 设置繁琐，消费级定位限制了高级工作流
- **发展方向不明确**：收购后产品感觉"少了紧迫感，少了客户导向"

### 2.5 优缺点总结

| 优点 | 缺点 |
|------|------|
| 被动数据丰富行业领先 | 收购后发展方向不明 |
| Automattic 背书，不会轻易消失 | 无 Android App |
| 免费 1,000 联系人 | 客服响应慢 |
| 关系强度评分实用 | 功能相对简单，无 AI 对话 |
| iOS App 质量高 | 消费级定位限制了高级功能 |
| $10/月定价亲民 | 团队功能有限（最多 4 人） |

## 三、Dex vs Clay 对比

| 维度 | Dex | Clay/Mesh |
|------|-----|-----------|
| **核心优势** | LinkedIn 深度集成 + AI 会前简报 | 被动数据丰富 + 关系强度评分 |
| **平台** | iOS + Android + 桌面 + Web | iOS + Mac + Web（无 Android） |
| **定价** | $12/月起（年付） | $10/月起（年付），免费 1,000 人 |
| **AI 能力** | 会前简报、Copilot 自然语言查询 | 关系评分、自然语言搜索 |
| **集成** | LinkedIn、Gmail、Outlook、Calendar | Gmail、Outlook、LinkedIn、Twitter、iCloud、iMessage、WhatsApp |
| **适合人群** | LinkedIn 重度用户、职业社交 | 投资人/VC、被动数据丰富需求 |
| **最大痛点** | Bug 多、同步慢、导入差 | 收购后沉寂、无 Android |
| **团队支持** | 不支持 | Team 计划（最多 4 人） |
