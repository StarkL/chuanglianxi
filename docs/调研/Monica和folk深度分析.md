# Monica 和 folk 深度分析

> 调研日期：2026-04-16
> 产品类别：开源代表 + 团队 CRM 代表

---

## 一、Monica（monicahq.com）

### 1.1 基本信息

| 项目 | 详情 |
|------|------|
| 定位 | 开源个人关系管理（PRM），隐私优先 |
| 创始时间 | 2017 年 2 月（François Briatte 创建，后移交社区） |
| 技术栈 | PHP / Laravel / Vue.js |
| 许可证 | AGPL-3.0 |
| GitHub | 24,459 Stars，2,440 Forks，40 位贡献者，95 个版本 |
| 最新开发 | v5.0.0-beta.5（2025-04-21），v4.x 为稳定版 |
| 平台 | **仅 Web**（响应式设计，无原生移动 App） |
| 用户数 | 官网显示 44,123 注册用户 |

### 1.2 功能清单

| 功能 | 详情 |
|------|------|
| 联系人档案 | 姓名、生日、纪念日、关系类型（朋友/家人/同事等） |
| 关系映射 | 定义联系人之间的关系（谁认识谁） |
| 交互日志 | 记录通话、会面、活动 |
| 笔记 | 为每个联系人添加笔记 |
| 自动生日提醒 | 按日程自动提醒 |
| 自定义联系提醒 | 设置定期联系提醒 |
| 礼物管理 | 记录送礼/收礼情况 |
| 债务管理 | 记录谁欠谁钱 |
| 日记 | 每日心情记录 |
| 活动类型自定义 | 自定义活动分类 |
| 标签系统 | 用标签组织联系人 |
| 多 Vault/用户 | 支持多个独立数据空间 |
| 文件/照片上传 | 存储文档和照片 |
| 自定义性别 | 支持自定义性别选项 |
| 收藏联系人 | 标记重要联系人 |
| 联系历史时间线 | 按时间查看所有交互 |

### 1.3 定价

| 版本 | 价格 | 说明 |
|------|------|------|
| 自托管 | **免费** | 完全免费，需自己维护服务器 |
| Monica Cloud Free | $0 | 功能有限，联系人数量受限 |
| Monica Cloud Premium | $9/月（$90/年） | 全功能解锁 |

### 1.4 技术架构

- **后端**：PHP 8.x + Laravel
- **前端**：Vue.js
- **数据库**：MySQL/MariaDB
- **部署**：Docker 支持，标准 LAMP 栈
- **API**：REST API（有限使用）
- **扩展性**：无插件生态系统

### 1.5 用户评价

**好评：**
- "个人关系上下文深度无人能及"——生活事件、对话、活动、提醒的组合
- "隐私优先的设计哲学"——自托管选项真正可用
- "提醒系统最实用"——设置一次，自动运行
- "简洁"——功能聚焦，UI 干净，上手快
- "社区活跃"——GitHub 活跃，功能路线由社区驱动
- "对 Asperger 综合征、阿尔茨海默症患者、内向者特别有用"

**差评：**
- **无 AI 功能**：2026 年这是越来越大的劣势，所有内容手动输入
- **仅 Web**：无原生移动 App，响应式设计但非移动优化
- **无导入功能**：除 Google Contacts 外无自动化导入器
- **无管道/交易追踪**：明确不做商业 CRM 功能
- **无工作流自动化**：无邮件营销、无自动触发
- **扩展性有限**：无插件生态，API 很少被用于集成
- **自然语言查询不可用**："3 个月没联系、住旧金山的搞技术的人" 这类查询需导出数据手动筛选
- **开发放缓**：v5 beta 开发超过一年，v4 稳定但更新慢

### 1.6 优缺点总结

| 优点 | 缺点 |
|------|------|
| 开源免费（自托管） | 无 AI 功能 |
| 隐私保护最好 | 仅 Web，无移动 App |
| 个人关系上下文最深 | 手动录入一切 |
| 提醒系统实用 | 导入功能差 |
| $9/月托管版便宜 | 无工作流自动化 |
| 社区驱动 | 开发放缓 |
| 简洁聚焦 | 扩展性差 |

## 二、folk（folk.app）

### 2.1 基本信息

| 项目 | 详情 |
|------|------|
| 定位 | 创始人/投资人/小团队的关系管理工具 |
| 平台 | Web（**无移动 App**） |
| Chrome 插件 | folkX（ standout 功能） |
| 评分 | Efficient App 4/5（效率评分） |

### 2.2 功能清单

| 功能 | 详情 |
|------|------|
| 电子表格式界面 | 类 Airtable 视图，直观 |
| Chrome 插件（folkX） | LinkedIn/任何网页一键添加联系人，行业最佳体验 |
| 自动数据丰富 | LinkedIn 数据 + 邮箱地址自动补充 |
| Magic Fields | AI 提示词自动生成信息 |
| 邮件序列 | 批量发送/自动发送邮件 |
| 管道管理 | 基础看板视图（非专业级） |
| 团队协作 | 多用户支持 |
| REST API | 新发布，功能较基础 |
| Zapier 集成 | 6,000+ 应用连接 |
| Gmail/Outlook 集成 | 邮箱同步 |
| WhatsApp 集成 | 需配置 |
| LinkedIn 集成 | 直接同步 |
| 联系人导入 | Chrome 插件 + CSV |
| 权限管理 | 细粒度访问控制 |

### 2.3 定价

| 计划 | 价格（每用户/月） | 核心功能 |
|------|-------------------|----------|
| Starter | $18-24 | 统一联系人、LinkedIn/邮箱同步、1 键丰富、邮件序列 |
| Team | $29-49 | 多渠道、AI 助手、管道、团队角色、分析 |
| Growth | 定制报价 | 大规模协作、优先支持、自定义集成、高级自动化 |

年付有折扣（ Efficient App 提供 10% 独家折扣码 FOLKXEFFICIENT）。

### 2.4 用户评价

**好评：**
- "Chrome 插件是市场上最好的联系人导入体验"
- "从 Google Sheets/Airtable 升级到 CRM 的最佳选择"
- "UI 美观，上手快"
- "AI 丰富覆盖基础需求"
- "小团队（2-10 人）协作好"

**差评：**
- **管道功能有限**：无概率权重、无收入预测、无活动销售提示
- **AI 不够智能**：不能和 CRM 对话（"给我看所有投资圈的人" 不可用）
- **API 早期**：功能基础，不适合技术型用户的高级需求
- **无移动 App**：Web only
- **按席位定价**：小团队还可以，扩大后成本高（3 人 Premium = $120/月 = $1,440/年）
- **数据在云端**：无自托管选项
- **Zapier 连接器有限制**：某些字段类型不支持

### 2.5 优缺点总结

| 优点 | 缺点 |
|------|------|
| Chrome 插件最佳 | 无移动 App |
| UI 美观 | 管道功能弱 |
| 小团队友好 | 按席位定价，扩展贵 |
| AI 丰富实用 | AI 不够智能（无自然语言查询） |
| API + Zapier 可扩展 | API 功能基础 |
| 快速上手 | 数据仅在 folk 云端 |

## 三、Monica vs folk 对比

| 维度 | Monica | folk |
|------|--------|------|
| **核心定位** | 个人关系管理（家人/朋友） | 小团队商业关系管理（投资人/创始人） |
| **开源** | AGPL-3.0，完全开源 | 闭源，SaaS |
| **定价** | 免费自托管 / $9/月托管 | $18-49/用户/月 |
| **移动 App** | 无（响应式 Web） | 无 |
| **AI** | 无 | 基础（Magic Fields、数据丰富） |
| **集成** | 极少 | Gmail、Outlook、LinkedIn、WhatsApp、Zapier |
| **适合** | 隐私优先的个人用户 | 创始人/投资人小团队 |
| **最大优势** | 开源 + 隐私 + 个人关系深度 | Chrome 插件 + 电子表格 UI |
| **最大劣势** | 无 AI + 手动录入 | 贵 + 无移动 App + 管道弱 |
